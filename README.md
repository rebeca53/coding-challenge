# coding-challenge

The questions solutions are:
1. Replacing characters in place: replace_char.cpp
2. Check words with jumbled letters: partial_permutation.py
3. Check words with typos: word_typo.py
4. [Android] Search on a list: SearchApplication
5. Remove duplicates on email thread: remove_duplicates.cpp
6. [Android] Email processor service: EmailProcessor
7. Linked List Intersection: list_intersection.cpp

The time and space complexity are explained in the comments of the code.
